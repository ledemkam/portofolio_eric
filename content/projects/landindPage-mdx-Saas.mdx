## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone https://github.com/ledemkam/landingPageSaas.git
cd landingPageSaas
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:5173`.

## Deployment

To deploy this project, you can use netlify or vercel. You can link your GitHub repository
to Vercel and deploy the project with a single click.
