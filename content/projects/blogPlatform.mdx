This is a news blogplatform website built with Reactjs ,Typescript,
Tailwind,Spring Boot Framework. 
## Features


- **Responsive Design**: The website is responsive and works on all devices.

## Technologies

- **Reactjs**: A React framework for building static and dynamic websites.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **HeroUI**: A ui-library for React that helps you build beautiful and accessible user interfaces.
- **Spring Boot**:  AN open-source Java framework used for programming standalone
- **Docker**: A freie Software zur Isolierung von Anwendungen mit Hilfe von Containervirtualisierung.
- **Postgresql**: A  powerful, open source object-relational database system.


## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone https://github.com/ledemkam/blogplatform.git

cd blogplatform :
   - cd blog-frontend
      npm install 
    Once the dependencies are installed, you can run the development server:
      npm run dev
      The development client will start at `http://localhost:5173`
   - cd blog-backend
     
      run docker
      run the backend application with the editor(eclipse or intellij)
      The development server will start at `http://localhost:8080`



```
